# Before Page Expected Weaknesses

The GEO platform is expected to detect these weaknesses in the before page:

- unclear category definition
- weak answerability
- thin FAQ
- no comparison table
- no trust block
- no pricing / limitations
- no schema
- no llms.txt
- weak citation readiness
- weak prompt coverage
- weak target audience detail
- no detailed use-case sections
- no AI platform testing workflow explanation
- no prompt-to-page gap mapping explanation
- no evidence folder explanation
- no clear statement of what the tool cannot guarantee
- no source-like facts that help citation systems

This is intentional. The page should be weak, but not broken.
