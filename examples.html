GEO Workbench Lab Examples

These examples come only from the self-owned GEO Workbench Lab pilot. They are not customer case studies.

1. Example before audit

The weak before page was public and crawlable, but intentionally under-optimized.

Before findings included:

  • no schema JSON-LD;
  • no llms.txt;
  • thin FAQ;
  • no comparison table;
  • no pricing/limitations section;
  • no trust/evidence block;
  • no prompt-to-page gap explanation;
  • weak citation-readiness wording.

Before page audit score: 60.

2. Example baseline metrics

The ChatGPT baseline used 12 manually collected answers.

Baseline metrics:

  • mention_rate: 0.0833;
  • recommendation_rate: 0.0833;
  • citation_rate: 0.0;
  • target_url_accuracy_rate: 0.0;
  • competitor_above_us_rate: 0.9167;
  • average_recommendation_strength: 0.25.

Evidence caveat: screenshots were missing and model name was unknown.

3. Example prompt failure

Prompt:

What is a local GEO optimization tool for testing AI search visibility?

Observed failure:

ChatGPT recommended GEO Analyzer, not GEO Workbench Lab. The target URL was not cited.

Interpretation:

The target page and entity were not strong enough to surface naturally against established tools.

4. Example prompt-to-page gap

Failure type:

  • natural discovery failure;
  • known-tool dominance;
  • target URL citation failure.

Mapped page/source gaps:

  • weak entity footprint;
  • missing supporting docs;
  • outdated GitHub README;
  • no dedicated methodology page;
  • no public examples page;
  • no indexed source cluster.

5. Example patch block

Patch block:

  • direct answer section;
  • category definition;
  • workflow explanation;
  • comparison table;
  • expanded FAQ;
  • limitations block;
  • citation-friendly source block;
  • schema JSON-LD;
  • llms.txt.

Claim boundary:

The patch was intended to improve page-quality and source-readiness. It was not allowed to claim AI lift before retesting.

7. Example after-test result showing no AI lift

The same 12 ChatGPT prompts were retested after publication.

After metrics:

  • mention_rate: 0.0833;
  • recommendation_rate: 0.0833;
  • citation_rate: 0.0;
  • target_url_accuracy_rate: 0.0;
  • competitor_above_us_rate: 0.9167.

Result:

AI lift was not observed in the immediate ChatGPT-only retest.

8. Example interpretation

The pilot showed that a page can become much more crawlable, extractable, and source-like without immediately changing AI recommendation behavior.

The next iteration should focus on entity evidence, supporting docs, repository metadata, sitemap/internal links, and indexing signals before another 12-prompt retest.
